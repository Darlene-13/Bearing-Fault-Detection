# Smart Hydraulic Support System

A smart hydraulic roof support system for small-scale underground mines in Kenya — combining mechanical support with real-time sensor monitoring and early-warning alerts.

## Project Overview

ASM underground mines rely on timber props that decay and fail without warning. This system replaces them with hydraulic supports that actively monitor roof pressure and convergence, alerting miners before a collapse occurs.

## Tech Stack

| Component | Technology |
|---|---|
| Microcontroller | ESP32-WROOM-32 |

| Build system | PlatformIO 6.1.19 |
| Simulator | Wokwi for VS Code |
